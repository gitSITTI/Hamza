🔐Security Design

Everything private stays on your machine. Cloudflare only proxies auth, never touches your data.

🔐 Cloudflare Zero Trust Tunnel (Free)

  • Install cloudflared on your machine
  • Creates a named tunnel — no public IP ever exposed
  • Cloudflare Access enforces email OTP or your Google account
  • You get a URL like mcp.yourname.cloudflareaccess.com
  • Free for up to 50 users (you only need 1)

🛡️ Local Encryption

  • SQLite encrypted with sqlcipher — passphrase on open
  • Documents (PDFs, tax returns) stored with AES-256 at rest
  • API keys never in source code — stored in .env + keychain
  • All personal MCP tool calls logged with timestamp + caller
  • Kill switch: one command disables tunnel and zeroes active sessions

🔑 API Key Management

  • Per-tool, per-caller permission scopes
  • Read-only key issued to NetWorth GUI (can't write)
  • Write key only for you locally (or Claude Code when you allow)
  • Keys rotated monthly (script provided)

📦 Document Vault

  • Upload PDFs: tax returns, pay stubs, bank statements, leases
  • MCP parses them with pdf-parse → extracts numbers into SQLite
  • Original files stored encrypted, never sent to external APIs
  • Claude Code can read parsed numbers, not raw files unless you allow

⚖️ Real Estate ↔ Bitcoin Offset

The core idea: RE is inflation-hedged, illiquid, leveraged. BTC is deflation-hedged, liquid, volatile. Together they offset each other's weaknesses.

  • User sets "comfort % in BTC vs RE" — MCP tracks drift from target
  • Correlation analysis: when RE values dip (rising rates), BTC often independent
  • Liquidity score: RE equity locked vs BTC instantly liquid
  • Model: "If I put $X less in down payment and $X into BTC Roth — what happens at year 10?"
  • Tax drag comparison: RE depreciation vs Roth BTC zero tax
